Then, add the serial port setup in the global section (I recommend placing it right after the Bluetooth setup, around the `BT_ADDR` block):

```python
# =========================
# PHYSICAL UART (SERIAL) OUTPUT - GPIO 14/15
# =========================
try:
    ser = serial.Serial(
        port='/dev/serial0',      # or '/dev/ttyS0' on some Pi models
        baudrate=9600,            # Change if your receiver needs a different speed (common: 9600, 19200, 115200)
        parity=serial.PARITY_NONE,
        stopbits=serial.STOPBITS_ONE,
        bytesize=serial.EIGHTBITS,
        timeout=0.1,
        write_timeout=0.1
    )
    print("UART opened successfully on /dev/serial0")
except Exception as e:
    print(f"Failed to open UART: {e}")
    ser = None
```

**Note:**  
- On most modern Raspberry Pi OS, `/dev/serial0` is the alias for the primary UART on GPIO 14/15.  
- You must first enable the serial port in `raspi-config` → Interface Options → Serial Port → **No** to login shell, **Yes** to serial hardware. Reboot after changing.
